# Scalar Types

Scalar types are `i1`, signed integers `i8`/`i16`/`i32`/`i64`, unsigned integers `u8`/`u16`/`u32`/`u64`, and floats `f32`/`f64`. `i1` is boolean, not numeric.

Integer operations include `plus`, `minus`, `times`, `divided by`, `remainder`, bitwise operators, and shifts. Float operations include arithmetic and ordered comparisons for matching float types. Casts are explicit:

```inscription,check
To main, giving i32.
Let byte_value: u8 be 65.
Give byte_value as i32.
```
